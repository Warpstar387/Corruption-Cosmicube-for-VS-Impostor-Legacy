# Corruption Cosmicube for VS Impostor Legacy
Corruption skins and songs ported over to VS Impostor Legacy as well as skins from Corruption Reimagined.
Requires Impostor Legacy V1.1.0 or higher.
Corruption Mod owned by Phantom Fear.
Corruption Reimagined owned by KumoriiJade.

INSTALLATION INSTRUCTIONS:
1.Download the Mod from either this page or the GameBanana or Google Drive pages.
2.Extract the zip using 7-Zip or whatever you usually use for extracting ZIPS.
3.Move the extracted folder to the content folder of Impostor Legacy (remember must be using V1.1.0 or higher). Layout should be Impostor Legacy/content/corruption.
Should also have the Security DLC installed as well (not just because it's awesome) but because the Corruption Songs in freeplay uses the number after Security DLC.
